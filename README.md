## 系统采用技术或开源框架：

1. 前端模板：adminLTE
2. 后端技术：spring-boot,maven,jpa,themleaf
3. 数据库: mysql,neo4j,elasticsearch
4. 其他工具：sts,vscode

## 项目启动配置:

1. 系统安装jdk1.8，mysql5.7，maven3.3 neo4j ,elasticsearch,python

2. 新建数据库:application

3. 导入数据库脚本 sql/All.sql

4. 启动服务端：
       1. 执行 start/start-build.bat  
       2. 执行 start/start-all.bat 

访问 http://localhost
